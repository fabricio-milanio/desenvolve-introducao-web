Exercício CSS - Unidade de Medida

1. Tamanhos das Fontes

Tamanho do h1

Regra: html h1 { font-size: 2rem; }
Cálculo: 2 * 20px = 40px ✅

Tamanho do .principal

Regra: h1 > .principal { font-size: 1.4em; }
Cálculo: 1.4 * 40px (tamanho do pai, o h1) = 56px ✅

Tamanho do .secundario

Regra: html .secundario { font-size: 1.4rem; }
Cálculo: 1.4 * 20px = 28px ✅
